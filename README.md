# 欠勤率予測モデル（Employee Absenteeism Prediction）

## 概要

本プロジェクトでは、従業員の属性情報や勤務状況のデータを基に、欠勤の発生（または欠勤率）を予測するモデルを構築しました。

人事データの分析により、欠勤傾向を事前に把握することで、労務管理の最適化や業務効率の改善への活用を想定しています。

---

## 使用技術

* Python
* Pandas / NumPy
* scikit-learn
* Matplotlib / Seaborn
* Jupyter Notebook

---

## データ前処理

### 1. データクリーニング

* 欠損値の確認・処理
* 不要列の削除
* データ型の整形

### 2. 特徴量エンジニアリング

* カテゴリ変数のダミー変数化（One-Hot Encoding）
* 数値特徴量の標準化（StandardScaler）

### 3. データ分割

* 学習データ
* テストデータ

---

## モデル構成

* 問題設定：

  * 二値分類（欠勤あり / なし）
    または
  * 回帰（欠勤時間・欠勤率）

使用モデル：

* ロジスティック回帰（分類の場合）
* または回帰モデル（Notebook構成に応じて）

設定：

* 最適化：scikit-learn標準アルゴリズム
* 評価指標：

  * Accuracy（分類）
  * またはMSE / R²（回帰）

---

## 学習と評価

* 学習データでモデルを訓練
* テストデータで性能を評価
* 予測結果と実測値の比較
* 混同行列や評価指標によりモデル性能を確認

---

## 工夫した点

* 人事データ特有のカテゴリ変数を適切にエンコード
* 標準化によりモデル学習の安定化
* 特徴量選択を行い、予測に重要な要素を整理
* 学習・テスト分割による汎化性能の確認

## まとめ

本プロジェクトでは、従業員データを用いた欠勤予測モデルを構築し、データ前処理、特徴量設計、モデル学習、評価までの一連の機械学習パイプラインを実装しました。
ビジネス課題（人事・労務管理）への応用を想定した、実務志向のデータ分析スキルを示すプロジェクトです。
